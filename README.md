gen-callgraph
=============

gen-callgraph is a bash script to generate call graph from binary

Usage
-----

<pre>
gen-call-graph &lt;BINARY&gt;
</pre>

Requirements
------------

* bash
* readelf
* objdump
* c++filt
* dot
